Year 8 • Topic

Introduction to Python Programming

Python is a general-purpose programming language. We’ll use it to learn how programs store data, make decisions, and repeat steps — like a robot with slightly better handwriting.

What you will learn

Learning goals

• Use variables to store and update values
• Take input and produce clear output
• Use selection (if/elif/else) for decisions
• Use iteration (for/while) for repetition
• Write readable code (naming, indentation, comments)
• Debug common errors and explain fixes

Key vocabulary

Variable Data type String Integer Boolean Selection Iteration Syntax Logic error

“It doesn’t work” isn’t a bug report — we go from symptoms → cause → fix.

How you’ll show mastery

• Build small programs (quiz, calculator, simple game)
• Use selection + loops appropriately (not “just because”)
• Trace code and predict outputs
• Fix errors using evidence (not vibes)
