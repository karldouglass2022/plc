Year 7 • Topic

Developing vector graphics

Vector graphics are made from shapes and paths, so they stay sharp at any size. Perfect for logos, icons, and anything that shouldn’t turn into pixel soup.

What you will learn

Learning goals

• Use shapes and lines to build images
• Edit paths/nodes to refine designs
• Organise work using layers and grouping
• Choose colours, fonts and alignment with purpose
• Export graphics for different uses (web/print)

Key vocabulary

Vector Raster Path Node Stroke Fill Layer Group

If you can explain “vector vs raster” without saying “just because”, you’re winning.

How you’ll show mastery

• Create a logo/icon that meets a brief
• Apply consistent colour + typography choices
• Demonstrate clean layering and alignment
• Export correctly (size, file type, transparent background)
