Year 9 • Topic

Python programming with sequences of data

Sequences let you store and process multiple values: strings (characters) and lists (items). This is the step from “toy programs” to “useful programs”.

What you will learn

Learning goals

• Create and use lists and strings
• Access items by index (and avoid off-by-one chaos)
• Use loops to process sequences
• Use common sequence operations (len, in, slicing basics)
• Build simple algorithms (searching, counting, totals)
• Write code that’s readable and testable

Key vocabulary

Sequence List String Index Iteration len() Slice Accumulator

Indexing starts at 0. Yes, it’s weird. No, we’re not changing it.

How you’ll show mastery

• Write programs that store many values (scores, names, sensor readings)
• Loop through sequences to compute results (max, min, average, counts)
• Explain (in words) what an algorithm does step-by-step
• Test using different data to prove it works
